## 顺序修正

用户原计划第二天 11:30—13:00 写“返回大良老城区午餐”，随后 13:00—13:30 又写“驹荣北路→大良钟楼”，存在顺序冲突。原型按实际动线调整为：

`顺峰山公园 → 驹荣北路 → 大良钟楼 → 老城区午餐 → 酒店取行李`

不改变景点与交通方式，只调整时间表达，避免重复折返。

## 2026-07-24 交通复核修订

| 复核项 | 修订结果 |
|---|---|
| 甘坑站到深圳北站 | 修正为“10 号线往福田口岸方向至五和站，换乘 5 号线往赤湾方向至深圳北站”；原写“坂田站换乘”错误 |
| 百鸽笼站到深圳北站 | 5 号线往赤湾方向直达，无需换乘 |
| 深圳北到广州南 | “29 分钟”是最快车程，页面改为约 29—40 分钟，并提示以实际车次为准 |
| 广州南到酒店 | 45—55 分钟主要对应地铁段；高铁下车至酒店建议预留 60—75 分钟 |
| 酒店门店 | 提醒核对是否为清晖路、靠近大良钟楼的“佛山顺德清晖园景区店”，避免与“环市北路店”混淆 |
| 顺峰山公园 | 大良钟楼到驹荣北路为 3 号线 2 站；驹荣北路 A 口方向出站后仍需步行约 10—15 分钟 |
| 容桂渔人码头 | 打车时间由 15—25 分钟调整为较保守的 20—30 分钟，周末拥堵需额外预留 |
| 第二天午餐 | 补回用户原计划中的鱼生 |
| 行前清单 | 补充身份证 / 乘车证件 |
